# ChatGPTQuizTechonology

An interactive quiz application that uses OpenAI's GPT-3.5 to generate challenging yes/no questions about technology. Test your knowledge with dynamically generated questions that adapt to your skill level!

## Features

- 🤖 **AI-Generated Questions**: Each question is dynamically created by GPT-3.5, ensuring unlimited unique questions
- 💡 **Tech-Focused**: Questions are specifically designed for technology professionals
- ✅ **Yes/No Format**: Simple and intuitive binary choice interface
- 🎯 **Score Tracking**: Real-time scoring system to track your performance
- 🎨 **Modern UI**: Themed interface with customizable backgrounds
- ⚡ **Non-blocking Operations**: Threaded API calls prevent UI freezing

## Prerequisites

Before running the application, ensure you have the following installed:

- Python 3.7 or higher
- OpenAI API key
